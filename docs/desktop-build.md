# Desktop build (retired)

Tauri desktop releases are **no longer published**.

Install and run the local app instead:

```bash
npm install -g foxschema    # or Homebrew / Winget+npm — see INSTALL.md
foxschema
foxschema shortcut          # Fox Schema icon on your Desktop
```

- User install: [INSTALL.md](INSTALL.md)
- Maintainers publish: [PUBLISH.md](PUBLISH.md)
- Server deploy: [DEPLOYMENT.md](DEPLOYMENT.md)

The `apps/desktop` tree may remain for reference but is not a release path.
